# Galaxy10 Dataset

This directory should contain the Galaxy10 DECals dataset.

## Download Instructions

1. Download the dataset from: http://astro.utoronto.ca/~bovy/Galaxy10/Galaxy10_DECals.h5
2. Place the `Galaxy10_DECals.h5` file in this directory

## Dataset Information

- **Size**: ~2.5 GB
- **Images**: 17,736 galaxy images
- **Resolution**: 256Ã—256 pixels (RGB)
- **Classes**: 10 morphological categories
- **Format**: HDF5

## Class Labels

0. Disturbed Galaxies
1. Merging Galaxies
2. Round Smooth Galaxies
3. In-between Round Smooth Galaxies
4. Cigar Shaped Smooth Galaxies
5. Barred Spiral Galaxies
6. Unbarred Tight Spiral Galaxies
7. Unbarred Loose Spiral Galaxies
8. Edge-on Galaxies without Bulge
9. Edge-on Galaxies with Bulge
